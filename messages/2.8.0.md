# MarkdownTOC - 2.8.0

## Changes

- Allow regular expressions in 'id_replacements' Ref: #113

## Notice

Setting format has been changed.
You have to modify `id_replacements` section if you customised `MarkdownTOC.sublime-settings` by yourself.

old format:

```json
{
  "id_replacements": {
    "-": " ",
    "" : ["!","#","$"]
  }
}
```

new format:

```json
{
  "id_replacements": [
    {
      "pattern": "\\s",
      "replacement": "-"
    },
    {
      "pattern": "!|#|$",
      "replacement": ""
    }
  ]
}
```
